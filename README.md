# wine_analysis_knn
# KNN y Reglas de Asociación con Python

Proyecto práctico de Machine Learning y Data Mining utilizando algoritmos K-Nearest Neighbors (KNN), PCA y Apriori para análisis de datos y detección de patrones.

## 📌 Objetivo

Aplicar técnicas de aprendizaje supervisado y minería de datos para analizar relaciones entre variables, realizar predicciones y descubrir patrones frecuentes en conjuntos de datos.

## 🛠 Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Mlxtend
- Jupyter Notebook

## 📊 Temas desarrollados

### 🔹 K-Nearest Neighbors (KNN)
- Entrenamiento del modelo
- Predicción de valores
- Normalización de datos
- Búsqueda de vecinos cercanos

### 🔹 PCA (Principal Component Analysis)
- Reducción de dimensionalidad
- Visualización de datos en 2D
- Interpretación de componentes principales

### 🔹 Reglas de Asociación
- Algoritmo Apriori
- Market Basket Analysis
- Generación de reglas de asociación
- Detección de patrones frecuentes

## 📈 Resultados

Se analizaron similitudes entre observaciones, patrones de consumo y relaciones entre variables mediante técnicas de Machine Learning y minería de datos.

## 🚀 Aprendizajes

- Implementación de KNN con Scikit-learn
- Escalado y preparación de datos
- Visualización con PCA
- Descubrimiento de patrones frecuentes con Apriori
- Interpretación de reglas de asociación
